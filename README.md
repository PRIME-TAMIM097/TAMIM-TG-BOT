# 🤖 TAMIM BOT — Telegram

A Telegram bot with **Video Downloader** + **AI Chat** features.

## ✨ Features
- 📥 Download videos from YouTube, Facebook, TikTok, Instagram
- 💬 AI Chat powered by Mahmud's Gemini API
- 🔗 Auto-detect video links in messages

## 🚀 Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Run the bot
```bash
node index.js
```

## 📋 Commands
| Command | Description |
|---------|-------------|
| `/start` | Welcome message |
| `/help` | Help menu |
| `/dl <url>` | Download video |
| `/chat <msg>` | Chat with AI |

## 🔗 Auto Download
Just send any video link directly — the bot will detect and download it!

**Supported platforms:**
- YouTube
- Facebook  
- TikTok
- Instagram
